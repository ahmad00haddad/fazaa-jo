# فزعة (Fazaa) - النخوة الأردنية في تطبيق واحد 🇯🇴

**"فزعة"** ليس مجرد تطبيق لتقديم الخدمات، بل هو **رقمنة لقيمة اجتماعية أردنية أصيلة** وهي "النخوة". نحن نؤمن أن المساعدة العاجلة هي جزء من هويتنا، وتطبيق فزعة يربط بين من يحتاج المساعدة العاجلة وبين "النشامى" المستعدين لتقديمها في نفس المنطقة الجغرافية.

---

## 🎯 الفكرة والمشكلة التي نحلها
في كثير من الأحيان، تواجهنا مواقف طارئة لا تحتاج إلى خدمات مدفوعة أو شركات، بل تحتاج إلى شخص قريب لديه القدرة على المساعدة. يعتمد نجاح "فزعة" على **الكثافة المجتمعية (Network Density)** والثقة المتبادلة. 

**التطبيق يغطي حالات الفزعة الحقيقية مثل:**
*   🚗 **تعطّل السيارة على الطريق:** (بطارية فارغة، بنشر، نفاد وقود في طرق مقطوعة).
*   🩸 **الحاجة الطبية العاجلة:** (توصيل دواء لكبار السن، الحاجة لفصيلة دم نادرة لإنقاذ حياة).
*   🚙 **التوصيلة الطارئة:** (خصوصاً في المناطق أو الأوقات التي يضعف فيها توفر سيارات الأجرة).
*   ❄️ **الطقس القاسي:** (المساعدة في سحب السيارات العالقة بالثلوج أو السيول).
*   🔍 **فقدان / إيجاد:** (طفل تائه، حيوان أليف، أو مقتنيات شخصية).
*   🤝 **المساعدة المجتمعية:** (دعم كبار السن، مساعدة العائلات المحتاجة، أو فزعة في مناسبات كالقرى).
*   🚨 **الطوارئ العامة:** (حريق، حادث، أو الحاجة لشهود ومساعدة يدوية فورية).

## 🛡️ الخصوصية والأمان (ميزة تنافسية للسياق المحافظ)
ندرك تماماً أهمية الأمان والخصوصية في مجتمعنا، لذلك صممنا **نظام الخصوصية بين الجنسين**:
*   **الطلبات النسائية الخاصة (Female-Only Requests):** يمكن للمرأة إنشاء طلب مساعدة (مثل توصيلة أو مساعدة منزلية) وتخصيصه ليظهر **فقط** للمستخدمات الإناث الموثقات. 
*   **حماية تقنية صارمة (RLS):** هذا الإخفاء ليس مجرد واجهة مستخدم (UI)، بل هو مفروض على مستوى قواعد البيانات؛ مما يعني أن الطلبات النسائية غير موجودة إطلاقاً في البيانات التي تصل لحسابات الذكور.
*   **توثيق الهوية:** لا يمكن المشاركة في هذه الميزات الحساسة دون توثيق حقيقي لرقم الهاتف والملف الشخصي.

---

## 🌟 لماذا "فزعة"؟ (الميزات الرئيسية)
*   **خريطة حية (Live Map):** رؤية الطلبات القريبة منك جغرافياً في الوقت الفعلي لتتمكن من الوصول السريع.
*   **نظام ثقة متكامل (Trust System):** توثيق أرقام الهواتف، نظام تقييم بعد كل فزعة، وشارات (Badges) للمتطوعين النشطين لبناء مجتمع موثوق.
*   **إشعارات فورية (Real-time Notifications):** تنبيهات فورية عندما يحتاج شخص بالقرب منك إلى فزعة، أو عندما يقبل شخص نداءك.
*   **تجربة مستخدم أصلية (Native-feel UX):** تصميم مستوحى من التطبيقات العالمية مع لمسة هوية أردنية (ألوان دافئة، خط عربي أصيل)، وحركات وانتقالات سلسة (Skeleton loaders, Bottom sheets) لضمان تجربة سريعة وخالية من التعقيد.

## 💻 التقنيات المستخدمة (Tech Stack)
*   **Frontend:** React (Vite), TypeScript, Tailwind CSS, shadcn/ui, Framer Motion
*   **State & Data Fetching:** React Query
*   **Backend & Auth:** Supabase (PostgreSQL, Realtime, Storage, Edge Functions, Row Level Security)
*   **Routing:** React Router DOM
*   **Cross-Platform (PWA/Mobile):** Capacitor, Vite PWA

## 🛠️ إعداد المشروع محلياً

1. **استنساخ المستودع**
```sh
git clone https://github.com/ahmad00haddad/fazaa-jo.git
cd fazaa-jo
```

2. **تثبيت الحزم (Dependencies)**
```sh
npm install
# أو
bun install
```

3. **إعداد المتغيرات البيئية (Environment Variables)**
قم بنسخ ملف `.env.example` إلى `.env` وضع مفاتيح Supabase الخاصة بك:
```sh
cp .env.example .env
```

4. **تشغيل خادم التطوير**
```sh
npm run dev
# أو
bun run dev
```

سيتم تشغيل التطبيق على الرابط: `http://localhost:8080`.
